Se te dará un texto basado en el periodo de riego de una planta y debes devolver un JSON con la información del riego, resumiendo los días o las semanas en que debe regarse la planta. El JSON que tienes que crear tiene la clave watering_period que será un array. Cada objeto dentro de watering_period es la informacion para una estación, y tiene que haber solamente 4. Cada objeto tendrá los campos 'unit' que será la unidad de medida de la periodicidad y corresponde con 'days' o 'weeks'. El campo 'value' tendrá un número o un rango de números de la forma '1-2' que será las veces que se riega según la unidad de medida. Si la unidad de medida es 'days', un 'value' de 1 quiere decir que se tiene que regar una vez al día. Si la unidad es 'weeks' y 'value' 1, quiere decir que se tiene que regar una vez a la semana. Una vez que comiences el JSON, debes construirlo correctamente y solo con los campos que te indico.

El texto es el siguiente:
"${TEXT}"

Sigue el siguiente ejemplo:
{
	"watering_period": [
		{
		"value": "Un número que representa la frecuencia de riego. Puede ser un número o un rango con dos números y un guion entre ellos, pero nunca pongas aquí las unidades; por ejemplo, nunca pongas 'días' o 'semanas'",
		"unit": "La unidad del valor. Solo puede ser days o weeks",
		"season": "La estación. Los valores aquí deben ser 'spring', 'summer', 'fall' o 'winter'"
		}
	]
}

Devuelve solo el JSON y ninguna explicación.
